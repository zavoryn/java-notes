## 🔗 模块间串联关系

> 面试中能跨模块串联是重要加分项。以下从三个维度理解模块间关联。

### 维度一：Agent 全链路串联

```
用户输入 → Agent感知(1.基础概念)
         → 规划拆解(2.ReAct + CoT/ToT)
         → 知识检索(3.RAG)
         → 工具调用(4.MCP + 5.Skill)
         → 记忆读写(6.Memory)
         → 多Agent协作(7.Multi-Agent)
         → 结果校验(8.测试评估)
         → 返回用户
```

### 维度二：分层架构串联

```
┌──────────────────────────────────────────────┐
│  应用层：场景设计(9) / Multi-Agent(7)          │  ← 面试展示层
├──────────────────────────────────────────────┤
│  能力层：Skill(5) / MCP(4) / Memory(6)        │  ← 核心能力层
├──────────────────────────────────────────────┤
│  推理层：ReAct(2) / RAG(3)                    │  ← 推理增强层
├──────────────────────────────────────────────┤
│  基础层：Agent概念(1) / 大模型基础(0)           │  ← 知识底座
├──────────────────────────────────────────────┤
│  保障层：测试与评估(8)                          │  ← 质量保障
└──────────────────────────────────────────────┘
```

### 维度三：与 Java 后端技术的交叉点

| Agent 模块 | 对应后端技术 | 串联要点 |
|-----------|------------|---------|
| 大模型推理 | Spring AI / LangChain4j | 推理框架集成、流式响应(SSE/WebSocket) |
| 短期记忆 | Redis | Session 管理、滑动窗口、TTL 续期 |
| 语义记忆 | MySQL / Redis Hash | KV 存储、冷热分离 |
| 情节记忆 | 向量数据库 | Embedding、HNSW 索引、语义检索 |
| 混合检索 | Elasticsearch | BM25 + Dense Vector、RRF 融合 |
| 异步管线 | Kafka / RabbitMQ | 记忆提取异步化、削峰填谷 |
| 记忆微服务 | 微服务架构 | 独立部署、水平扩展、全链路 Trace |
| Agent 编排 | DAG 工作流 | 状态机、检查点、条件分支 |
| MCP 协议 | HTTP / gRPC | JSON-RPC 2.0、OAuth 2.1 PKCE |

## 🎯 面试复习优先级

### P0 必须掌握（每面必考 · 10+ 公司高频）
1. **RAG 完整流程**（索引→检索→增强→生成）+ Chunk 策略 + Embedding 原理
2. **Agent 核心组件与整体流程** + Agent vs 传统 LLM 本质区别
3. **Memory 四层记忆体系** + 记忆管线三阶段 + 幻觉防御
4. **大模型基础**：推理原理（Prefill/Decode/KV Cache）、训练范式（SFT/RLHF/DPO）
5. **ReAct 范式**（TAO 循环）+ Function Calling 机制

### P1 重点掌握（高频考题 · 5-10 公司高频）
6. MCP 协议与 Function Calling 区别
7. 向量数据库选型（Milvus / Pinecone / Weaviate）
8. 幻觉问题解决方案（三层防御）
9. Summarization vs Reflection
10. Skill 系统设计（when_to_use + 路由策略）
11. 场景设计题万能框架（5 步法）
12. 上下文工程 / 压缩 / 管理策略

### P2 加分项（区分度）
13. Multi-Agent 协作模式 + A2A 协议
14. GraphRAG 原理与适用场景
15. Agent 死循环检测与异常处理
16. Agentic RAG 设计
17. Agent 测试方法（Harness Engineering）
18. 记忆偏见与纠正
