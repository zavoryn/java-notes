# MySQL 总览

> [!summary] 模块速记
> **MySQL = 存储引擎(InnoDB) + 索引(B+树) + 事务(MVCC+锁) + 日志(redo/undo/binlog+主从) + 调优。**
> **一条主线：SELECT/UPDATE → 存储引擎 → 索引定位 → MVCC/加锁 → redo/undo/binlog → 主从同步。**
