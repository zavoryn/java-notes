# 多线程与并发编程总览

> [!summary] 模块速记
> **大厂面试多线程五大核心：JMM（可见性/有序性）、synchronized 锁升级、CAS（原子性）、AQS（Lock 体系基石）、线程池（生产核心）。**
> **一条主线串起来：多线程三大问题（可见性/原子性/有序性）→ 每种问题对应什么解决方案 → 解决方案的底层实现。**

## 三大问题 → 解决方案 → 底层实现（核心主线）

| 并发问题 | 描述 | 解决方案 | 底层机制 |
|----------|------|----------|----------|
| **可见性** | 一个线程改了，另一个线程看不到 | volatile / synchronized / Lock | JMM happens-before + 内存屏障 |
| **原子性** | `i++` 三步操作被打断 | synchronized / Lock / Atomic(CAS) | 锁 / CAS 循环 |
| **有序性** | 指令重排导致逻辑错乱 | volatile / synchronized / Lock | 内存屏障禁止重排序 |
