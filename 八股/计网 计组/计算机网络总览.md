# 计算机网络总览

> [!summary] 模块速记
> **计算机网络 = 体系结构(OSI/TCP/IP) + TCP连接管理(三次握手/四次挥手/TIME_WAIT) + TCP可靠传输(五板斧) + 传输层对比(TCP vs UDP vs QUIC) + HTTP演进(1.0→1.1→2.0→3.0) + HTTPS与TLS(混合加密+证书) + 网络应用(DNS/CDN/Cookie/JWT/跨域) + 综合面试(全链路+追问)。**
> **一条主线：从输入URL到页面展示 = DNS解析 → TCP握手 → TLS握手 → HTTP请求 → 服务端处理 → 浏览器渲染 → 连接关闭。**
