### StringBuilder和StringBuffer的区别是什么？
> 关联：[java基础](../../java基础/)

- StringBuilder：线程不安全，性能更高
- StringBuffer：线程安全，方法大多加了 synchronized，性能相对低
- 大多数实际开发中更常用 StringBuilder，因为字符串拼接通常发生在方法内部，是局部变量

### 遇到过哪些线程安全问题？
> 关联：[juc并发编程](../../juc并发编程/)

多个线程同时更新同一个统计值、库存数量、任务状态，如果直接用普通变量，会出现覆盖更新问题。例如 count++ 不是原子操作（读取、加1、写回），可以使用 AtomicInteger 来解决。

### ArrayList和LinkedList的区别？LinkedList是双向链表还是单向链表？
> 关联：[java基础](../../java基础/)

- ArrayList：基于动态数组实现，有扩容机制（1.5倍），元素在内存中连续分布
- LinkedList：基于双向链表实现，理论上没有大小限制，元素在内存中分散分布

### 讲一下HashMap？怎么根据hash值定位到数组？一定要重写hashCode和equals吗？
> 关联：[java基础](../../java基础/)

- JDK7：链表+数组，JDK8：链表+数组+红黑树
- hash & (n - 1) 定位数组位置（等价于取模，但更高效）
- 如果把自定义对象作为 key，必须重写 hashCode 和 equals

### 讲一下怎么实现单例模式？
> 关联：[java基础](../../java基础/)

- 静态内部类单例
- 双重检查锁单例（注意 volatile 防止指令重排序）
- 饿汉式

### Java 类的执行顺序
> 关联：[java基础](../../java基础/)

- 类第一次被使用：父类静态变量/静态代码块 -> 子类静态变量/静态代码块
- 对象创建时：父类实例变量初始化 -> 父类构造方法 -> 子类实例变量初始化 -> 子类构造方法

### 用过哪些Java命令？遇到过实际问题吗？
> 关联：[jvm虚拟机](../../jvm虚拟机/)

- jps：查看 Java 进程
- jstack：查看线程栈（排查死锁、卡顿）
- jmap：查看内存（dump 堆、分析对象）

### 怎么在Linux上查看Java进程占用端口？
> 关联：[jvm虚拟机](../../jvm虚拟机/)

jps -l 拿到 PID，netstat -tlnp | grep PID 获取对应端口。

### 怎么在Linux上查看Java进程打开了哪些文件？
> 关联：[jvm虚拟机](../../jvm虚拟机/)

lsof -p PID

### 怎么在Linux上查看Java进程内存占用率？
> 关联：[jvm虚拟机](../../jvm虚拟机/)

top -p PID

### 讲一下输入URL到返回网页的过程
> 关联：[计算机网络](../../计算机网络/)

输入URL -> DNS解析 -> TCP建立连接（可能TLS） -> 发送HTTP请求 -> 服务端处理 -> 返回响应 -> 浏览器解析+渲染 -> 连接复用或关闭

### DNS使用的端口？DNS怎么找到根域名服务器？本地DNS配置写在哪里？
> 关联：[计算机网络](../../计算机网络/)

- DNS默认端口53，常见DNS：Google 8.8.8.8、CloudFlare 1.1.1.1
- 递归解析 + 迭代查询：本地DNS -> 根服务器(.) -> .com顶级域 -> 权威DNS
- Linux本地DNS配置：/etc/resolv.conf
