# Redis

## Redis基础 ★★★★★ 全部是重点！！！

1. Redis的概念、特点 (内存数据库、高性能、支持多种数据结构、持久化等)
2. Redis的应用场景 (缓存、分布式锁、消息队列、计数器、排行榜等)
3. Redis的数据结构及底层实现 (String、Hash、List、Set、Sorted Set)
4. Redis的过期键删除策略 (惰性删除、定期删除) 及内存淘汰机制 (LRU、LFU等)

## 持久化 ★★★★★ 全部是重点！！！

1. RDB的原理 (快照方式)、触发时机 (手动: save/bgsave；自动配置)、优缺点
2. AOF的原理 (记录命令)、同步策略 (appendfsync: always、everysec、no)、优缺点
3. RDB与AOF的区别及选择建议
4. AOF重写的原理与触发时机 (手动: bgrewriteaof；自动配置)
5. Redis的数据恢复流程 (RDB与AOF同时存在时的优先级)

## 缓存问题 ★★★★★ 缓存三剑客，重点！！！

1. 缓存穿透的概念、原因及解决方案 (布隆过滤器、空值缓存等)
2. 缓存击穿的概念、原因及解决方案 (互斥锁、热点数据永不过期等)
3. 缓存雪崩的概念、原因及解决方案 (过期时间随机化、集群部署、热点数据永不过期等)
4. 缓存与数据库的一致性问题 (更新策略: 先更新数据库再删缓存、延迟双删等)
5. 缓存的更新策略 (LRU、LFU等) 及适用场景

## 高可用 ★★★★★

主从复制、哨兵、集群

1. Redis主从复制的原理 (全量复制、增量复制)、作用及配置
2. 哨兵 (Sentinel) 的作用 (监控、通知、自动故障转移) 及工作原理
3. Redis Cluster集群的架构 (分片、主从复制)、数据分布方式 (哈希槽)
4. Redis Cluster的故障检测与恢复机制
5. Redis高可用方案的对比与选择

## 其他核心问题 ★★★★ 一般出现在追问

1. Redis的事务实现 (MULTI、EXEC、DISCARD、WATCH) 及特点 (不支持回滚)
2. Redis的分布式锁实现 (SET NX EX命令、Redlock算法) 及注意事项
3. Redis的管道 (Pipeline) 机制及作用 (减少网络开销)
4. Redis 6.0的新特性 (多线程IO、ACL权限控制等)
5. Redis的内存优化策略 (数据结构选择、过期键清理等)
