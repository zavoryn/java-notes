# MySQL

## 数据库基础

1. MySQL 架构（连接层、服务层、引擎层、存储层）(概率低，选择性学习)
2. 常见的存储引擎（InnoDB、MyISAM）的区别（事务、锁、索引、适用场景）
3. 事务的 ACID 特性及实现原理（原子性：undo log；一致性：业务逻辑；隔离性：锁；持久性：redo log）
4. 事务的隔离级别（读未提交、读已提交、可重复读、串行化）及对应问题（脏读、不可重复读、幻读）
5. MySQL 的默认隔离级别，如何实现可重复读？（MVCC）

## 索引
b+数和hash索引

1. 索引的概念、作用及优缺点
2. B+树索引、哈希索引的结构（叶子节点有序且相连）及查询原理，为何不用B树或红黑树？
3. 索引的分类（主键索引、二级索引、联合索引、唯一索引）
4. 联合索引的最左匹配原则，如何合理设计联合索引？
5. 索引失效的场景（函数操作、类型转换、不等于、模糊查询左匹配等）
6. 覆盖索引、回表查询、索引下推的概念及作用
7. 哈希索引的特点及适用场景，InnoDB 的自适应哈希索引
8. 索引设计的原则（适合索引的字段，避免过度索引等）

## SQL 优化

1. Explain 命令的使用（各字段含义：id、select_type、type、key、rows 等）
2. 常见 SQL 语句的优化（SELECT、INSERT、UPDATE、DELETE）
3. 子查询与关联查询（JOIN）的优化，JOIN 的类型及区别（概率低，选择性学习）
4. 排序优化（filesort 的产生与解决）
5. 分页查询优化（limit 大偏移量问题，如何解决？）
6. 避免索引失效的编写规范

## 锁机制

1. MySQL 锁的分类（表锁、行锁、页锁）
2. InnoDB 的行锁类型（共享锁 S、排他锁 X）及加锁方式
3. 意向锁（意向共享锁 IS、意向排他锁 IX）的作用
4. 间隙锁（Gap Lock）、临键锁（Next-Key Lock）的产生原因及作用（解决幻读）
5. 死锁的产生与解决（预防、检测、处理）
6. 乐观锁与悲观锁的实现方式及适用场景（版本号、CAS）

## 日志与恢复

1. MySQL常见日志（binlog、redo log、undo log）的作用与区别
2. binlog 的格式（STATEMENT、ROW、MIXED）及用途（主从复制、数据恢复）(概率低，选择性学习)
3. redo log 与 binlog 的关系，两阶段提交的原理（保证事务一致性）
4. MySQL 的数据恢复方式（基于 binlog 恢复、基于备份恢复）
