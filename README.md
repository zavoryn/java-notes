# Java 面试备战笔记

> 系统化的 Java 后端面试准备资料库，涵盖 Java 基础、JVM、并发编程、Spring 生态、数据库、中间件、分布式系统、算法数据结构等多个维度。基于 Obsidian 组织，支持双向链接与知识图谱。

## 内容导航

### 八股文（基础理论）

| 专题 | 内容 |
|------|------|
| **Java 基础** | 面向对象、异常机制、集合框架、泛型、注解、反射、IO/NIO |
| **Java 集合** | HashMap & ConcurrentHashMap 源码分析、ArrayList、LinkedList |
| **JVM** | 内存模型、类加载机制、字节码、GC 算法与调优、性能诊断 |
| **多线程** | 线程池原理与调优、锁机制（synchronized/AQS/ReentrantLock）、volatile、ThreadLocal |
| **Spring Boot** | 自动配置原理、启动流程、条件装配、监控端点 |
| **SSM 框架** | Spring IoC & AOP 深度解析、Spring MVC 请求链路、MyBatis 插件机制 |
| **MySQL** | 索引原理（B+Tree）、查询优化、事务与锁、MVCC、分库分表 |
| **Redis** | 数据结构、持久化、缓存策略、分布式锁、集群方案 |
| **消息队列** | Kafka vs RocketMQ 对比、消息可靠性、顺序消息、事务消息 |
| **微服务** | CAP 理论、服务发现、配置中心、分布式事务（Seata/TCC）、限流熔断 |
| **设计模式** | 单例、工厂、策略、观察者、代理等常用模式及 Spring 中的应用 |
| **操作系统** | 进程/线程、虚拟内存、文件系统、IO 模型 |
| **计算机网络** | TCP/IP、HTTP/HTTPS、WebSocket、DNS、CDN |
| **Elasticsearch** | 倒排索引、搜索优化、聚合分析 |

### 算法

按题型分类的 LeetCode 刷题记录，包含：
- **数组与字符串**、**链表**、**哈希表**、**双指针**
- **栈与队列**、**二叉树**、**堆（优先队列）**
- **回溯算法**、**二分查找**、**动态规划**
- **贪心算法**、**滑动窗口**、**图论**

每道题包含题目分析、多种解法对比、时空复杂度分析及踩坑记录。

### 面试专题

- **系统设计** — 秒杀系统、IM 系统、Feed 流、短链系统、排行榜
- **开放题** — 线上问题排查、系统优化思路、职业规划
