## API 备忘录

| 场景 | ✅ 正确 | ❌ 错误 |
|------|------|------|
| 栈 | `Deque<T> s = new ArrayDeque<>()` | `Stack<T> s = new Stack<>()` |
| 栈入/出 | `push()` / `pop()` / `peek()` | |
| 优先队列入/出 | `offer()` / `poll()` / `peek()` | `add()` / `pop()` |
| Set 查存在 | `set.contains(x)` | ~~`set.containsKey(x)`~~ |
| Set 建集合 | `new HashSet<>()` | |
| Map 查存在 | `map.containsKey(x)` | |
| Map 取或建 | `map.computeIfAbsent(k, v->new X())` | 手动 if-null-put |
| Java substring | `s.substring(start, end)` (end exclusive!) | 当成 inclusive |
| `nextInt()` 后 | 必须 `sc.nextLine()` 吃换行 | 直接 `nextLine()` |
| 数组复制 | `System.arraycopy(src, srcPos, dest, destPos, len)` | 手动 for |
| 数组排序 | `Arrays.sort(arr)` | |
| 拆分+转整数 | `Integer.parseInt(s.split(" ")[1])` | |
| 列表转数组 | `list.toArray(new int[0][])` | |
