# 每日温度

> [LeetCode 739. Daily Temperatures](https://leetcode.cn/problems/daily-temperatures/)

## 题目

给定一个整数数组 `temperatures`，表示每天的温度，返回一个数组 `answer`，其中 `answer[i]` 是指对于第 `i` 天，**下一个更高温度**出现在几天后。如果之后没有更高的温度，则 `answer[i] = 0`。

```
输入：temperatures = [73,74,75,71,69,72,76,73]
输出：[1,1,4,2,1,1,0,0]
```

## ACM 输入格式

```
第一行：n（天数）
第二行：n 个整数（温度）
```

```
8
73 74 75 71 69 72 76 73
```

## 解答

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] t = new int[n];
        for (int i = 0; i < n; i++) t[i] = sc.nextInt();

        int[] res = dailyTemperatures(t);
        for (int i = 0; i < n; i++) {
            if (i > 0) System.out.print(" ");
            System.out.print(res[i]);
        }
        sc.close();
    }

    public static int[] dailyTemperatures(int[] t) {
        int n = t.length;
        int[] ans = new int[n];
        Deque<Integer> stack = new ArrayDeque<>(); // 存下标，栈底→栈顶 递减

        for (int i = 0; i < n; i++) {
            while (!stack.isEmpty() && t[i] > t[stack.peek()]) {
                int prev = stack.pop();
                ans[prev] = i - prev;
            }
            stack.push(i);
        }
        // 剩下的下标 ans 默认为 0
        return ans;
    }
}
```

## 复杂度

- **时间**：O(n) — 每个元素入栈出栈各一次
- **空间**：O(n)

## 踩坑记录

> ❓ 单调栈存值还是存下标？
>
> **存下标。** 这题要算 "几天后"，是下标差 `i - prev`。存值只能判断大小，无法算距离。

> ⚠️ **`while` 条件不能漏 `!stack.isEmpty()`**：必须先判栈非空再 peek，否则空栈直接抛 `NoSuchElementException`。
> ```java
> // ❌ 直接 peek 可能空栈
> while (t[i] > t[stack.peek()])
> // ✅ 先判非空
> while (!stack.isEmpty() && t[i] > t[stack.peek()])
> ```
