# 柱状图中最大的矩形

> [LeetCode 84. Largest Rectangle in Histogram](https://leetcode.cn/problems/largest-rectangle-in-histogram/)

## 题目

给定 n 个非负整数，表示柱状图中每个柱子的高度。每个柱子宽度为 1。求柱状图中能勾勒出的最大矩形面积。

```
输入：heights = [2,1,5,6,2,3]
输出：10（5和6组成的 5×2=10）
```

## ACM 输入格式

```
第一行：n
第二行：n 个整数
```

## 解答

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] heights = new int[n];
        for (int i = 0; i < n; i++) heights[i] = sc.nextInt();
        System.out.println(largestRectangleArea(heights));
        sc.close();
    }

    public static int largestRectangleArea(int[] heights) {
        int n = heights.length;
        int[] h = new int[n + 2]; // 首尾加 0 做哨兵
        System.arraycopy(heights, 0, h, 1, n);

        Deque<Integer> stack = new ArrayDeque<>();
        int maxArea = 0;
        for (int i = 0; i < h.length; i++) {
            while (!stack.isEmpty() && h[i] < h[stack.peek()]) {
                int height = h[stack.pop()];
                int width = i - stack.peek() - 1;
                maxArea = Math.max(maxArea, height * width);
            }
            stack.push(i);
        }
        return maxArea;
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(n)

## 踩坑记录

> ❓ 为什么首尾加 0？
>
> 首 0 保证栈永不为空（不用判 !isEmpty）；尾 0 保证遍历结束时所有柱子都被弹出计算。

## 复习标记

> 🔁 2026-05-16 思路遗忘——单调递增栈+哨兵。关键理解：以某柱子高度画矩形，遇到比它矮的柱子就画不过去了，所以左右边界就是"第一个比它矮的柱子"，宽度 = 右边界 - 左边界 - 1
