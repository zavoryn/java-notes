# 数据流的中位数

> [LeetCode 295. Find Median from Data Stream](https://leetcode.cn/problems/find-median-from-data-stream/)

## 题目

实现 MedianFinder 类：`addNum(num)` 添加数字，`findMedian()` 返回当前所有数字的中位数。

```
输入：
addNum(1), addNum(2) → findMedian() = 1.5
addNum(3) → findMedian() = 2.0
```

## 解答

```java
static class MedianFinder {
    PriorityQueue<Integer> left = new PriorityQueue<>((a, b) -> b - a);  // 大顶堆：较小的一半
    PriorityQueue<Integer> right = new PriorityQueue<>();                 // 小顶堆：较大的一半

    public void addNum(int num) {
        left.offer(num);
        right.offer(left.poll());        // 先放左边，再把左边最大的移到右边
        if (left.size() < right.size())  // 平衡：左边可以多一个
            left.offer(right.poll());
    }

    public double findMedian() {
        if (left.size() > right.size()) return left.peek();
        return (left.peek() + right.peek()) / 2.0;
    }
}
```

## 复杂度

- **时间**：O(log n) per addNum
- **空间**：O(n)

## 踩坑记录

> 大顶堆用 `(a,b) -> b-a`，小顶堆用默认。奇数时左堆多一个（左堆顶就是中位数）。
