# 寻找两个正序数组的中位数

> [LeetCode 4. Median of Two Sorted Arrays](https://leetcode.cn/problems/median-of-two-sorted-arrays/)

## 题目

给定两个大小分别为 m 和 n 的正序数组 nums1 和 nums2，找出并返回这两个正序数组的中位数。要求 O(log(m+n))。

```
输入：nums1 = [1,3], nums2 = [2]
输出：2.0
```

## 解答

```java
class Solution {
    public double findMedianSortedArrays(int[] nums1, int[] nums2) {
        //对短的数组二分查找
        if(nums1.length>nums2.length){
            return findMedianSortedArrays(nums2,nums1);
        }
        int m=nums1.length;
        int n=nums2.length;
        int left=0;
        //数组切分的位置，所以是m
        int right=m;

        while(left<=right){
            //i和j是两个数组的切分点
            //需要保证两个切分点左侧的和右侧的数量
            int i=left+(right-left)/2;
            int j=(m+n+1)/2-i;

            //考虑切分点的极限情况
            int nums1LeftMax=(i==0)?Integer.MIN_VALUE:nums1[i-1];
            int nums1RightMin=(i==m)?Integer.MAX_VALUE:nums1[i];
            int nums2LeftMax=(j==0)?Integer.MIN_VALUE:nums2[j-1];
            int nums2RightMin=(j==n)?Integer.MAX_VALUE:nums2[j];

            if(nums1LeftMax<=nums2RightMin && nums2LeftMax<=nums1RightMin){
                //找到正确切分位置
                if((m+n)%2==1){
                    return Math.max(nums1LeftMax,nums2LeftMax);
                }else{
                    return (Math.max(nums1LeftMax,nums2LeftMax)+Math.min(nums1RightMin,nums2RightMin))/2.0;
                }
            }else if( nums1LeftMax>nums2RightMin){
                //偏大，往左边移动
                right=i-1;
            }else{
                //偏小,向右移动
                left=i+1;
            }
        }
        throw new IllegalArgumentException("not valid");
    }
}
```

## 复杂度

- **时间**：O(log min(m, n))
- **空间**：O(1)

## 复习标记

> 🔁 2026-05-13 重点复习：① 分割线含义——i切nums1、j切nums2，左半个数=(m+n+1)/2；② 判定条件——`nums1LeftMax<=nums2RightMin && nums2LeftMax<=nums1RightMin`；③ 极限用MIN/MAX不影响比较；④ `while(left<=right)` 闭区间写法，找到即return
> 🔁 2026-05-14 复习通过。新理解：① 中位数为什么取 max(leftMax)/avg(leftMax,rightMin)——分割线把所有元素切成两半，奇数左多1个取左半最大，偶数左右一样多取平均；② `(m+n+1)/2` 的+1是向上取整，让左半始终>=右半
> 🔁 2026-05-20 代码多处错误：①i/j是"切几刀"不是索引，左半最后一个=nums1[i-1]，右半第一个=nums1[i]，边界i==0用MIN,i==m用MAX；②二分更新用right=i-1/left=i+1，不是left--/right++；③while用left<=right闭区间
