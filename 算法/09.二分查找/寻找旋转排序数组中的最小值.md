# 寻找旋转排序数组中的最小值

> [LeetCode 153. Find Minimum in Rotated Sorted Array](https://leetcode.cn/problems/find-minimum-in-rotated-sorted-array/)

## 题目

已知一个长度为 n 的升序数组，旋转 1 到 n 次。找出数组中的最小元素。

```
输入：nums = [3,4,5,1,2]
输出：1
```

## 解答

```java
public static int findMin(int[] nums) {
    int left = 0, right = nums.length - 1;
    while (left < right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] > nums[right]) left = mid + 1;
        else right = mid;
    }
    return nums[left];
}
```

## 复杂度

- **时间**：O(log n)
- **空间**：O(1)

## 复习标记

> 🔁 2026-05-13 曾不理解为什么循环结束就是结果——本质是二分排除法，每次排除一半不可能含答案的区间，`left==right` 时只剩唯一解
