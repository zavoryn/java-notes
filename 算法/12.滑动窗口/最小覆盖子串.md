# 最小覆盖子串

> [LeetCode 76. Minimum Window Substring](https://leetcode.cn/problems/minimum-window-substring/)

## 题目

给你一个字符串 s、一个字符串 t。返回 s 中涵盖 t 所有字符的最小子串。如果 s 中不存在涵盖 t 所有字符的子串，则返回空字符串 ""。

```
输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"
```

## 解答

```java
public static String minWindow(String s, String t) {
    int[] need = new int[128];
    for (char c : t.toCharArray()) need[c]++;

    int left = 0, right = 0, count = t.length();
    int minStart = 0, minLen = Integer.MAX_VALUE;

    while (right < s.length()) {
        if (need[s.charAt(right++)]-- > 0) count--;
        while (count == 0) {
            if (right - left < minLen) { minStart = left; minLen = right - left; }
            if (need[s.charAt(left++)]++ == 0) count++;
        }
    }
    return minLen == Integer.MAX_VALUE ? "" : s.substring(minStart, minStart + minLen);
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(1)

## 复习标记

> 🔁 2026-05-20 代码4处错误：①minLen初始0应为Integer.MAX_VALUE，否则第一次比较永远不成立；②收缩while循环不应嵌套在`if(valid++)`里，要和if平级——因为收缩后valid可能不变但while还要继续；③收缩时没维护valid——移出字符后若curWin<count要valid--；④收缩时没判need.containsKey直接取curWin导致NPE
