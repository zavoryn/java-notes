# 环形链表

> [LeetCode 141. Linked List Cycle](https://leetcode.cn/problems/linked-list-cycle/) ｜ [牛客 NC4 ACM 练习](https://www.nowcoder.com/practice/650474f313294468a4ded3ce0f7898b9)

## 题目

给你一个链表的头节点 `head`，判断链表中是否有环。

```
输入：head = [3,2,0,-4], pos = 1
输出：true
解释：链表中有一个环，其尾部连接到第二个节点
```

## ACM 输入格式

```
第一行：n（节点个数）+ 环入口位置 pos（-1 表示无环）
第二行：n 个整数（节点值）
```

```
4 1
3 2 0 -4
```

## 解答

```java
import java.util.*;

public class Main {
    static class ListNode {
        int val;
        ListNode next;
        ListNode(int x) { val = x; }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int pos = sc.nextInt();

        ListNode[] nodes = new ListNode[n];
        for (int i = 0; i < n; i++) {
            nodes[i] = new ListNode(sc.nextInt());
        }
        for (int i = 0; i < n - 1; i++) {
            nodes[i].next = nodes[i + 1];
        }
        if (pos >= 0 && pos < n) {
            nodes[n - 1].next = nodes[pos];
        }

        System.out.println(hasCycle(n > 0 ? nodes[0] : null));
        sc.close();
    }

    public static boolean hasCycle(ListNode head) {
        ListNode slow = head, fast = head;
        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
            if (slow == fast) return true;
        }
        return false;
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(1)
