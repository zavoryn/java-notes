# 反转链表

> [LeetCode 206. Reverse Linked List](https://leetcode.cn/problems/reverse-linked-list/) ｜ [牛客 NC78 ACM 练习](https://www.nowcoder.com/practice/75e878df47f24fdc9dc3e400ec6058ca)

## 题目

给你单链表的头节点 `head`，请你反转链表，并返回反转后的链表。

```
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]
```

## ACM 输入格式

```
第一行：n（节点个数）
第二行：n 个整数（节点值）
```

```
5
1 2 3 4 5
```

## 解答

```java
import java.util.*;

public class Main {
    static class ListNode {
        int val;
        ListNode next;
        ListNode(int x) { val = x; }
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        ListNode dummy = new ListNode(0);
        ListNode cur = dummy;
        for (int i = 0; i < n; i++) {
            cur.next = new ListNode(sc.nextInt());
            cur = cur.next;
        }

        ListNode newHead = reverseList(dummy.next);

        while (newHead != null) {
            System.out.print(newHead.val);
            if (newHead.next != null) System.out.print(" ");
            newHead = newHead.next;
        }
        sc.close();
    }

    public static ListNode reverseList(ListNode head) {
        ListNode prev = null;
        ListNode curr = head;
        while (curr != null) {
            ListNode next = curr.next;
            curr.next = prev;
            prev = curr;
            curr = next;
        }
        return prev;
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(1)

## 踩坑记录

> ❓ 内部类能不能不加 `static`，直接用 `class ListNode {}`？
>
> **不能。** `main` 是 static 方法，非静态内部类 `ListNode` 必须依附于外部类 `Main` 的实例才能创建。static 上下文里拿不到这个外部实例，`new ListNode()` 直接编译报错。
>
> ```java
> // ❌ 编译错误：non-static variable this cannot be referenced from a static context
> class ListNode { ... }
>
> // ✅ 正确写法（三选一）
> static class ListNode { ... }     // 方案1：静态嵌套类（推荐）
> // 或把 class ListNode 放到 Main 外面  // 方案2：顶层类（但不能用 public）
> ```
>
> **本质**：`static class` 不依赖外部实例，本质就是一个独立顶层类，只是放在 `Main` 里做命名空间隔离。

> ❓ 输出末尾多了空格？
>
> ```java
> // ❌ 末尾多余空格，部分 OJ 判格式错误
> System.out.print(result.val + " ");
>
> // ✅ 先打值，再判断后面要不要加空格
> System.out.print(result.val);
> if (result.next != null) System.out.print(" ");
> ```

> ⚠️ **建表时游标和哨兵不要混用同一个变量**
>
> ```java
> // ❌ 同一个变量既当哨兵又当游标
> ListNode head = new ListNode(0);
> for (...) { head.next = ...; head = head.next; }
> head = head.next; // 此时 head 是最后一个节点，.next = null！
>
> // ✅ 哨兵和游标分开
> ListNode dummy = new ListNode(0);
> ListNode cur = dummy;
> for (...) { cur.next = ...; cur = cur.next; }
> reverseList(dummy.next); // 用哨兵拿真正的头
> ```
