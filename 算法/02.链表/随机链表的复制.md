# 随机链表的复制

> [LeetCode 138. Copy List with Random Pointer](https://leetcode.cn/problems/copy-list-with-random-pointer/)

## 题目

给你一个长度为 n 的链表，每个节点除了 next 指针还有一个 random 指针，可指向链表中任意节点或 null。构造这个链表的深拷贝。

## 解答

```java
public static Node copyRandomList(Node head) {
    if (head == null) return null;
    Map<Node, Node> map = new HashMap<>();
    Node cur = head;
    while (cur != null) { map.put(cur, new Node(cur.val)); cur = cur.next; }
    cur = head;
    while (cur != null) {
        map.get(cur).next = map.get(cur.next);
        map.get(cur).random = map.get(cur.random);
        cur = cur.next;
    }
    return map.get(head);
}
```

> 两遍：第一遍 HashMap 创建所有节点，第二遍连接 next 和 random。O(n) 时间 O(n) 空间。
