# 最长回文子串

> [LeetCode 5. Longest Palindromic Substring](https://leetcode.cn/problems/longest-palindromic-substring/) ｜ [牛客 NC17 ACM 练习](https://www.nowcoder.com/practice/b4525d1d84934cf280439aeecc36f4af)

## 题目

给你一个字符串 `s`，找到 `s` 中最长的回文子串。

```
输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案
```

## ACM 输入格式

```
单行一个字符串
```

```
babad
```

## 解答

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String s = sc.nextLine();

        System.out.println(longestPalindrome(s));
        sc.close();
    }

    public static String longestPalindrome(String s) {
        int len=s.length();
        int maxStart=0;
        int maxLen=1;

        for(int i=0;i<len;i++){
            for(int j=0;j<=1;j++){
                int left=i;
                int right=i+j;
                while(left>=0 && right<len && s.charAt(left)==s.charAt(right)){
                    left--;
                    right++;
                }
                left++;
                right--;

                if(maxLen<right-left+1){
                    maxLen=right-left+1;
                    maxStart=left;
                }
            }
        }
        return  s.substring(maxStart,maxStart+maxLen);
    }
}
```

## 复杂度

- **时间**：O(n²) — 每个字符扩展一次
- **空间**：O(1)

## 踩坑记录

> ❓ `expand` 返回值 `right - left - 1` 怎么来的？
>
> 退出循环时 left 和 right 多走了一步（上次匹配成功后移了，这次才发现不匹配）。回文实际区间是 `[left+1, right-1]`，长度 = `(right-1) - (left+1) + 1 = right - left - 1`。
>
> ```
> 以 i=1 为中心扩 "bab"：
> 初始 left=1, right=1 → 匹配后 left=0, right=2 → 匹配后 left=-1, right=3 → 越界退出
> 实际回文[0,2] "bab"，长度 = 3 - (-1) - 1 = 3 ✓
> ```

> ❓ `start = i - (len - 1) / 2` 和 `end = i + len / 2` 怎么理解？
>
> 以中心 `i` 为基准，回文向左延伸 `(len-1)/2` 个，向右延伸 `len/2` 个。`int` 整除让奇偶统一：
>
> ```
> 奇数 len=3: start = i - (3-1)/2 = i-1 ✓   end = i + 3/2 = i+1 ✓
> 偶数 len=4: start = i - (4-1)/2 = i-1 ✓   end = i + 4/2 = i+2 ✓
> ```
>
> 偶数左边的字符比右边少一个（中心在 i 和 i+1 之间，i 是左中心），所以 `(len-1)/2` 取地板，`len/2` 取天花板。

> ⚠️ **`substring(start, end)` 第二个参数是 exclusive！**
>
> ```java
> // ❌ 少 1 个字符
> return s.substring(start, end);
> // ✅ Java substring 的 end 是开区间
> return s.substring(start, end + 1);
> ```
