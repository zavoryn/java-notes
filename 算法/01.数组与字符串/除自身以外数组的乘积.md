# 除自身以外数组的乘积

> [LeetCode 238. Product of Array Except Self](https://leetcode.cn/problems/product-of-array-except-self/)

## 题目

给你一个整数数组 nums，返回数组 answer，其中 answer[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积。不要使用除法。

```
输入：nums = [1,2,3,4]
输出：[24,12,8,6]
```

## 解答

```java
public static int[] productExceptSelf(int[] nums) {
    int n = nums.length;
    int[] res = new int[n];
    res[0] = 1;
    for (int i = 1; i < n; i++) res[i] = res[i - 1] * nums[i - 1];  // 前缀积
    int right = 1;
    for (int i = n - 1; i >= 0; i--) { res[i] *= right; right *= nums[i]; } // 乘后缀积
    return res;
}
```
