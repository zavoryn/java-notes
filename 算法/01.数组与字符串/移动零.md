# 移动零

> [LeetCode 283. Move Zeroes](https://leetcode.cn/problems/move-zeroes/)

## 题目

给定一个数组 nums，将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。必须原地操作。

```
输入：nums = [0,1,0,3,12]
输出：[1,3,12,0,0]
```

## 解答

```java
public static void moveZeroes(int[] nums) {
    int slow = 0;
    for (int fast = 0; fast < nums.length; fast++) {
        if (nums[fast] != 0) {
            int tmp = nums[slow];
            nums[slow] = nums[fast];
            nums[fast] = tmp;
            slow++;
        }
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(1)
