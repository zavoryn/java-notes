# 不同路径

> [LeetCode 62. Unique Paths](https://leetcode.cn/problems/unique-paths/)

## 题目

一个机器人位于一个 m x n 网格的左上角。机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角。问总共有多少条不同的路径？

```
输入：m = 3, n = 7
输出：28
```

## 解答

```java
public static int uniquePaths(int m, int n) {
    int[] dp = new int[n];
    Arrays.fill(dp, 1);
    for (int i = 1; i < m; i++)
        for (int j = 1; j < n; j++)
            dp[j] += dp[j - 1];
    return dp[n - 1];
}
```
