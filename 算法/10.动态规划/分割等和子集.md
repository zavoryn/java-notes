# 分割等和子集

> [LeetCode 416. Partition Equal Subset Sum](https://leetcode.cn/problems/partition-equal-subset-sum/)

## 题目

给你一个只包含正整数的非空数组 nums。判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

```
输入：nums = [1,5,11,5]
输出：true（[1,5,5] 和 [11] 和都是 11）
```

## 解答

```java
public static boolean canPartition(int[] nums) {
    int sum = Arrays.stream(nums).sum();
    if (sum % 2 != 0) return false;
    int target = sum / 2;
    boolean[] dp = new boolean[target + 1];
    dp[0] = true;
    for (int num : nums)
        for (int j = target; j >= num; j--)
            dp[j] |= dp[j - num];
    return dp[target];
}
```

## 复杂度

- **时间**：O(n × target)
- **空间**：O(target)
