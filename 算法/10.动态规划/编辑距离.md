# 编辑距离

> [LeetCode 72. Edit Distance](https://leetcode.cn/problems/edit-distance/)

## 题目

给你两个单词 word1 和 word2，请返回将 word1 转换成 word2 所使用的最少操作数。可以插入、删除、替换一个字符。

```
输入：word1 = "horse", word2 = "ros"
输出：3（horse→rorse→rose→ros）
```

## 解答

```java
public static int minDistance(String word1, String word2) {
    int m = word1.length(), n = word2.length();
    int[] dp = new int[n + 1];
    for (int j = 0; j <= n; j++) dp[j] = j;
    for (int i = 1; i <= m; i++) {
        int prev = dp[0]; dp[0] = i;
        for (int j = 1; j <= n; j++) {
            int tmp = dp[j];
            if (word1.charAt(i - 1) == word2.charAt(j - 1)) dp[j] = prev;
            else dp[j] = 1 + Math.min(prev, Math.min(dp[j], dp[j - 1]));
            prev = tmp;
        }
    }
    return dp[n];
}
```
