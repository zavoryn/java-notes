# 乘积最大子数组

> [LeetCode 152. Maximum Product Subarray](https://leetcode.cn/problems/maximum-product-subarray/)

## 题目

给你一个整数数组 nums，请你找出数组中乘积最大的非空连续子数组，并返回该子数组的乘积。

```
输入：nums = [2,3,-2,4]
输出：6（2×3）
```

## 解答

```java
public static int maxProduct(int[] nums) {
    int max = nums[0], imax = nums[0], imin = nums[0];
    for (int i = 1; i < nums.length; i++) {
        if (nums[i] < 0) { int tmp = imax; imax = imin; imin = tmp; }
        imax = Math.max(nums[i], imax * nums[i]);
        imin = Math.min(nums[i], imin * nums[i]);
        max = Math.max(max, imax);
    }
    return max;
}
```

## 踩坑记录

> 负数会让最小变最大。所以同时维护 `imax` 和 `imin`。遇到负数先交换两者。
