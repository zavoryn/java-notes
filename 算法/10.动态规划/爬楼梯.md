# 爬楼梯

> [LeetCode 70. Climbing Stairs](https://leetcode.cn/problems/climbing-stairs/) ｜ [牛客 NC68](https://www.nowcoder.com/practice/c5b75e8ade6242e6a6e5c8e6e1c1c1c1)

## 题目

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

```
输入：n = 3
输出：3（1+1+1, 1+2, 2+1）
```

## ACM 输入格式

```
一行：n
```

```
3
```

## 解答

### 解法一：标准 DP（一维数组）

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        System.out.println(climbStairs(n));
        sc.close();
    }

    public static int climbStairs(int n) {
        if (n <= 2) return n;
        int[] dp = new int[n + 1];
        dp[1] = 1;
        dp[2] = 2;
        for (int i = 3; i <= n; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
}
```

### 解法二：滚动变量（空间 O(1)）

```java
public static int climbStairs(int n) {
    if (n <= 2) return n;
    int prev2 = 1, prev1 = 2;
    for (int i = 3; i <= n; i++) {
        int cur = prev1 + prev2;
        prev2 = prev1;
        prev1 = cur;
    }
    return prev1;
}
```

## 复杂度

| 解法 | 时间 | 空间 |
|------|------|------|
| 标准 DP | O(n) | O(n) |
| 滚动变量 | O(n) | O(1) |

## 踩坑记录

> ❓ 这题是不是就是斐波那契数列？
>
> 是的，`dp[i] = dp[i-1] + dp[i-2]`，和斐波那契完全一致。唯一的区别是起始值：`dp[1]=1, dp[2]=2`（斐波那契是 `F(1)=1, F(2)=1`）。
