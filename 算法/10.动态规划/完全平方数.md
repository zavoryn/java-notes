# 完全平方数

> [LeetCode 279. Perfect Squares](https://leetcode.cn/problems/perfect-squares/)

## 题目

给定正整数 n，找到若干个完全平方数（1,4,9,16...）使得它们的和等于 n。你需要让组成和的完全平方数个数最少。

```
输入：n = 12
输出：3（4+4+4）
```

## 解答

```java
public static int numSquares(int n) {
    int[] dp = new int[n + 1];
    Arrays.fill(dp, n + 1);
    dp[0] = 0;
    for (int i = 1; i <= n; i++)
        for (int j = 1; j * j <= i; j++)
            dp[i] = Math.min(dp[i], dp[i - j * j] + 1);
    return dp[n];
}
```
