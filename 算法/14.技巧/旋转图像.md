# 旋转图像

> [LeetCode 48. Rotate Image](https://leetcode.cn/problems/rotate-image/)

## 题目

给定一个 n×n 的二维矩阵，原地将其顺时针旋转 90 度。

```
输入：matrix = [[1,2,3],[4,5,6],[7,8,9]]
输出：[[7,4,1],[8,5,2],[9,6,3]]
```

## 解答

```java
public static void rotate(int[][] matrix) {
    int n = matrix.length;
    for (int i = 0; i < n; i++)                    // 沿主对角线翻转
        for (int j = i + 1; j < n; j++)
            swap(matrix, i, j, j, i);
    for (int i = 0; i < n; i++)                    // 左右翻转
        for (int j = 0; j < n / 2; j++)
            swap(matrix, i, j, i, n - 1 - j);
}
```

> 顺时针 90° = 主对角线翻转 + 左右翻转。

## 复习标记

> 🔁 2026-05-13 对角线遍历 `j=i+1` 的含义遗忘——只遍历主对角线上方，每对元素只交换一次；上下翻转+对角线翻转 和 对角线翻转+左右翻转 效果相同
