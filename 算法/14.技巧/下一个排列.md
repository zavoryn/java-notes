# 下一个排列

> [LeetCode 31. Next Permutation](https://leetcode.cn/problems/next-permutation/)

## 题目

整数数组的下一个排列是指其整数的下一个字典序更大的排列。原地修改，只使用常数额外空间。如果不存在下一个更大的排列，则重排为最小序列（升序）。

```
输入：nums = [1,2,3]
输出：[1,3,2]
```

## 解答

```java
public static void nextPermutation(int[] nums) {
    int i = nums.length - 2;
    while (i >= 0 && nums[i] >= nums[i + 1]) i--;       // 找到第一个降序
    if (i >= 0) {
        int j = nums.length - 1;
        while (nums[j] <= nums[i]) j--;                   // 找到比它大的最小数
        swap(nums, i, j);
    }
    reverse(nums, i + 1, nums.length - 1);                // 翻转后半段
}
static void reverse(int[] a, int l, int r) { while (l < r) swap(a, l++, r--); }
```

> 三步：① 从右找第一个 `a[i] < a[i+1]` ② 从右找第一个 > a[i] 的交换 ③ 翻转 i+1 到末尾。
