# 颜色分类

> [LeetCode 75. Sort Colors](https://leetcode.cn/problems/sort-colors/)

## 题目

给定一个包含 0,1,2 共 n 个元素的数组，原地排序。不能使用库排序函数。

```
输入：nums = [2,0,2,1,1,0]
输出：[0,0,1,1,2,2]
```

## 解答

```java
public static void sortColors(int[] nums) {
    int p0 = 0, p2 = nums.length - 1, i = 0;
    while (i <= p2) {
        if (nums[i] == 0) swap(nums, i++, p0++);
        else if (nums[i] == 2) swap(nums, i, p2--);
        else i++;
    }
}
static void swap(int[] a, int i, int j) { int t = a[i]; a[i] = a[j]; a[j] = t; }
```

> 荷兰国旗：三指针，p0 维护 0 边界，p2 维护 2 边界。换 2 时 i 不前进（换过来的值未检查）。
