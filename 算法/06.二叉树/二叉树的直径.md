# 二叉树的直径

> [LeetCode 543. Diameter of Binary Tree](https://leetcode.cn/problems/diameter-of-binary-tree/)

## 题目

二叉树的直径是任意两个节点路径长度中的最大值。这条路径可能穿过也可能不穿过根节点。

```
输入：root = [1,2,3,4,5]
输出：3（路径 [4,2,1,3] 或 [5,2,1,3]）
```

## 解答

```java
static int maxDiameter = 0;

public static int diameterOfBinaryTree(TreeNode root) {
    maxDiameter = 0;
    depth(root);
    return maxDiameter;
}

static int depth(TreeNode root) {
    if (root == null) return 0;
    int left = depth(root.left);
    int right = depth(root.right);
    maxDiameter = Math.max(maxDiameter, left + right); // 经过当前节点的最长路径
    return 1 + Math.max(left, right); // 当前节点为根的深度
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(h)
