# 二叉树的层序遍历

> [LeetCode 102. Binary Tree Level Order Traversal](https://leetcode.cn/problems/binary-tree-level-order-traversal/) ｜ [牛客 NC15](https://www.nowcoder.com/practice/04a5560e43e24e9db4595865dc9c63a3)

## 题目

给你二叉树的根节点 `root`，返回其节点值的**层序遍历**（即逐层地，从左到右访问所有节点）。

```
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
```

## ACM 输入格式

```
第一行：n
接下来 n 行：val left right
```

```
5
3 9 20
9 -1 -1
20 15 7
15 -1 -1
7 -1 -1
```

## 解答

```java
import java.util.*;

public class Main {
    static class TreeNode {
        int val;
        TreeNode left, right;
        TreeNode(int x) { val = x; }
    }

    static Map<Integer, TreeNode> map = new HashMap<>();

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        TreeNode root = null;
        for (int i = 0; i < n; i++) {
            int v = sc.nextInt(), l = sc.nextInt(), r = sc.nextInt();
            TreeNode node = map.computeIfAbsent(v, TreeNode::new);
            if (i == 0) root = node;
            if (l != -1) node.left = map.computeIfAbsent(l, TreeNode::new);
            if (r != -1) node.right = map.computeIfAbsent(r, TreeNode::new);
        }

        List<List<Integer>> res = levelOrder(root);
        System.out.print("[");
        for (int i = 0; i < res.size(); i++) {
            if (i > 0) System.out.print(",");
            System.out.print(res.get(i));
        }
        System.out.println("]");
        sc.close();
    }

    public static List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if (root == null) return res;

        Queue<TreeNode> q = new LinkedList<>();
        q.offer(root);
        while (!q.isEmpty()) {
            int size = q.size();
            List<Integer> level = new ArrayList<>();
            for (int i = 0; i < size; i++) {
                TreeNode node = q.poll();
                level.add(node.val);
                if (node.left != null) q.offer(node.left);
                if (node.right != null) q.offer(node.right);
            }
            res.add(level);
        }
        return res;
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(n) — 队列最坏存整层节点

---

## 踩坑记录

> ❓ 为什么用 `q.size()` 先存下来？
>
> 不存的话，`for` 循环里 `q.offer` 会让队列不断变大，`q.size()` 动态增长，无法区分层边界。先取 `int size = q.size()` 固定当层节点数。
