# 对称二叉树

> [LeetCode 101. Symmetric Tree](https://leetcode.cn/problems/symmetric-tree/) ｜ [牛客 NC16](https://www.nowcoder.com/practice/ff05d44dfdb04e1d83bdbdab320efbcb)

## 题目

给定一个二叉树，检查它是否是**轴对称**的。

```
输入：root = [1,2,2,3,4,4,3]
输出：true

输入：root = [1,2,2,null,3,null,3]
输出：false
```

## ACM 输入格式

```
第一行：n
接下来 n 行：val left right
```

```
7
1 2 2
2 3 4
2 4 3
3 -1 -1
4 -1 -1
4 -1 -1
3 -1 -1
```

## 解答

```java
import java.util.*;

public class Main {
    static class TreeNode {
        int val;
        TreeNode left, right;
        TreeNode(int x) { val = x; }
    }

    static Map<Integer, TreeNode> map = new HashMap<>();

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        TreeNode root = null;
        for (int i = 0; i < n; i++) {
            int v = sc.nextInt(), l = sc.nextInt(), r = sc.nextInt();
            TreeNode node = map.computeIfAbsent(v, TreeNode::new);
            if (i == 0) root = node;
            if (l != -1) node.left = map.computeIfAbsent(l, TreeNode::new);
            if (r != -1) node.right = map.computeIfAbsent(r, TreeNode::new);
        }

        System.out.println(isSymmetric(root));
        sc.close();
    }

    public static boolean isSymmetric(TreeNode root) {
        if (root == null) return true;
        return check(root.left, root.right);
    }

    static boolean check(TreeNode p, TreeNode q) {
        if (p == null && q == null) return true;
        if (p == null || q == null) return false;
        return p.val == q.val
            && check(p.left, q.right)
            && check(p.right, q.left);
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(h)

## 踩坑记录

> ❓ 为什么不能"中序遍历后判断回文"？
>
> 中序序列相同 ≠ 镜像对称（如 `[1,2,2,2,null,2]`）。必须同时比较左右子树的镜像关系。
