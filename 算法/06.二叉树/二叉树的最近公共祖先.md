# 二叉树的最近公共祖先

> [LeetCode 236. Lowest Common Ancestor of a Binary Tree](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/) ｜ [牛客 NC102](https://www.nowcoder.com/practice/e0d4cfb12fce4e71a2138e0eb5d04f9c)

## 题目

给定一个二叉树，找到该树中两个指定节点的**最近公共祖先**（LCA）。

```
输入：root = [3,5,1,6,2,0,8,null,null,7,4], p = 5, q = 1
输出：3
```

## ACM 输入格式

```
第一行：n
值）接下来 n 行：val left right
最后一行：p q（两个目标节点
```

```
9
3 5 1
5 6 2
1 0 8
6 -1 -1
2 7 4
0 -1 -1
8 -1 -1
7 -1 -1
4 -1 -1
5 1
```

## 解答

```java
import java.util.*;

public class Main {
    static class TreeNode {
        int val;
        TreeNode left, right;
        TreeNode(int x) { val = x; }
    }

    static Map<Integer, TreeNode> map = new HashMap<>();

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        TreeNode root = null;
        for (int i = 0; i < n; i++) {
            int v = sc.nextInt(), l = sc.nextInt(), r = sc.nextInt();
            TreeNode node = map.computeIfAbsent(v, TreeNode::new);
            if (i == 0) root = node;
            if (l != -1) node.left = map.computeIfAbsent(l, TreeNode::new);
            if (r != -1) node.right = map.computeIfAbsent(r, TreeNode::new);
        }
        int pv = sc.nextInt(), qv = sc.nextInt();

        TreeNode ans = lowestCommonAncestor(root, map.get(pv), map.get(qv));
        System.out.println(ans.val);
        sc.close();
    }

    public static TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if (root == null || root == p || root == q) return root;
        TreeNode left = lowestCommonAncestor(root.left, p, q);
        TreeNode right = lowestCommonAncestor(root.right, p, q);
        if (left != null && right != null) return root;
        return left != null ? left : right;
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(h)

## 踩坑记录

> ❓ 为什么 `root == p || root == q` 时直接返回 root？
>
> 此时不需要继续向下找——如果另一个节点在子树里，当前就是 LCA；如果不在，返回当前节点作为"找到了"的标记传给上层。

## 四句话总结 LCA 递归

```
1. 终止：碰到 null / p / q → 返回当前节点
2. left = 在左子树找 p 或 q
3. right = 在右子树找 p 或 q
4. 两边都非 null → 当前是 LCA；否则传回非 null 那边
```
