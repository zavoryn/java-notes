# 验证二叉搜索树

> [LeetCode 98. Validate Binary Search Tree](https://leetcode.cn/problems/validate-binary-search-tree/)

## 题目

给你一个二叉树的根节点 root，判断其是否是一个有效的二叉搜索树。BST 定义：左子树所有节点值 < 根 < 右子树所有节点值。

```
输入：root = [2,1,3]
输出：true
输入：root = [5,1,4,null,null,3,6]
输出：false（4 < 5 但 4 在右子树）
```

## 解答

```java
public static boolean isValidBST(TreeNode root) {
    return check(root, Long.MIN_VALUE, Long.MAX_VALUE);
}

static boolean check(TreeNode root, long low, long high) {
    if (root == null) return true;
    if (root.val <= low || root.val >= high) return false;
    return check(root.left, low, root.val) && check(root.right, root.val, high);
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(h)

## 踩坑记录

> ❓ 为什么用 `Long` 而不是 `Integer`？
>
> 边界情况 `[2147483647]`（即 Integer.MAX_VALUE），用 `Integer` 做参数无法区分"没限制"和"限制等于 MAX/MIN"。
