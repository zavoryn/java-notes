# 二叉树的最大深度

> [LeetCode 104. Maximum Depth of Binary Tree](https://leetcode.cn/problems/maximum-depth-of-binary-tree/) ｜ [牛客 NC13](https://www.nowcoder.com/practice/435fb86331474282a3499955f0a41e8b)

## 题目

给定一个二叉树，找出其最大深度。二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

```
输入：root = [3,9,20,null,null,15,7]
输出：3
```

## ACM 输入格式

```
第一行：n（节点数）
接下来 n 行：val left right（-1 表示空）
```

```
5
3 9 20
9 -1 -1
20 15 7
15 -1 -1
7 -1 -1
```

## 解答

```java
import java.util.*;

public class Main {
    static class TreeNode {
        int val;
        TreeNode left, right;
        TreeNode(int x) { val = x; }
    }

    static Map<Integer, TreeNode> map = new HashMap<>();

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        TreeNode root = null;
        for (int i = 0; i < n; i++) {
            int v = sc.nextInt(), l = sc.nextInt(), r = sc.nextInt();
            TreeNode node = map.computeIfAbsent(v, TreeNode::new);
            if (i == 0) root = node;
            if (l != -1) node.left = map.computeIfAbsent(l, TreeNode::new);
            if (r != -1) node.right = map.computeIfAbsent(r, TreeNode::new);
        }

        System.out.println(maxDepth(root));
        sc.close();
    }

    public static int maxDepth(TreeNode root) {
        if (root == null) return 0;
        return 1 + Math.max(maxDepth(root.left), maxDepth(root.right));
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(h) — 递归栈深度

## 踩坑记录

> ❓ 空树返回 0 还是返回 -1？
>
> 这题空树返回 0（"节点数"定义）。但有些题（如平衡二叉树）空树高度定义为 -1，读题时注意。
