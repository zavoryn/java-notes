# 二叉树展开为链表

> [LeetCode 114. Flatten Binary Tree to Linked List](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/)

## 题目

给你二叉树的根结点 root，将它展开为一个单链表：展开后的单链表应该同样使用 TreeNode，其中 right 子指针指向链表中下一个结点，而左子指针始终为 null。展开后的单链表应该与二叉树先序遍历顺序相同。

## 解答
采用右左根的顺序搭建这个链表
```java
public static void flatten(TreeNode root) {
    TreeNode cur = prev;
    while (cur != null) {
       if (root == null) return;

        flatten(root.right);
        flatten(root.left);

        root.right = prev;
        root.left = null;
        prev = root;
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(1)
