# 二叉树的中序遍历

> [LeetCode 94. Binary Tree Inorder Traversal](https://leetcode.cn/problems/binary-tree-inorder-traversal/) ｜ [牛客 NC45](https://www.nowcoder.com/practice/1b25a41f25f2411588d2cd55f8c40f79)

## 题目

给定一个二叉树的根节点 `root`，返回它的**中序**遍历结果。

```
输入：root = [1,null,2,3]
输出：[1,3,2]
```

## ACM 输入格式

```
第一行：n（节点个数）
接下来 n 行：每行三个整数 val left right
  -1 表示空节点
```

```
3
1 -1 2
2 3 -1
3 -1 -1
```

## 解答

```java
import java.util.*;

public class Main {
    static class TreeNode {
        int val;
        TreeNode left, right;
        TreeNode(int x) { val = x; }
    }

    static Map<Integer, TreeNode> map = new HashMap<>();

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        TreeNode root = null;
        for (int i = 0; i < n; i++) {
            int v = sc.nextInt(), l = sc.nextInt(), r = sc.nextInt();
            TreeNode node = map.computeIfAbsent(v, TreeNode::new);
            if (i == 0) root = node;
            if (l != -1) node.left = map.computeIfAbsent(l, TreeNode::new);
            if (r != -1) node.right = map.computeIfAbsent(r, TreeNode::new);
        }

        List<Integer> res = inorderTraversal(root);
        for (int i = 0; i < res.size(); i++) {
            if (i > 0) System.out.print(" ");
            System.out.print(res.get(i));
        }
        sc.close();
    }

    public static List<Integer> inorderTraversal(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        Deque<TreeNode> stack = new ArrayDeque<>();
        TreeNode cur = root;
        while (cur != null || !stack.isEmpty()) {
            while (cur != null) {
                stack.push(cur);
                cur = cur.left;
            }
            cur = stack.pop();
            res.add(cur.val);
            cur = cur.right;
        }
        return res;
    }
}
```

## 复杂度

- **时间**：O(n)
- **空间**：O(h) — h 为树高，最坏 O(n)

## 踩坑记录

> ❓ 递归还是迭代？
>
> 面试常要求**迭代版**。关键是手动用栈模拟递归：一直往左走到底，弹出一个访问，再转右子树。

> ❓ `Map.computeIfAbsent` 是什么？建树时为什么要用 `map`？
>
> `map.computeIfAbsent(v, TreeNode::new)` 等价于：先查 `get(v)`，有就直接返回，没有才 `new TreeNode(v)` 并存进 map。
>
> **目的**：同一个值在整个建树过程中只创建一个 `TreeNode`。输入里一个节点会作为"当前行主角"出现一次，也会作为别人的 `left`/`right` 被提前创建——用 map 保证两次拿到的是同一个对象，否则树断链。
>
> `TreeNode::new` 是方法引用，等价于 `v -> new TreeNode(v)`。

> ❓ `map` 为什么要加 `static`？
>
> `main` 是 static 方法，字段不加 `static` 无法在 `main` 里直接使用。但也可以直接声明为 `main` 里的局部变量，不用 static 也行——笔记里写成 static 字段只是为了跨方法复用方便。
